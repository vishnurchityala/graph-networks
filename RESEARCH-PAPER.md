# Abstract

Multimodal meme classification is challenging because hateful or misogynistic intent often emerges from the interaction between textual and visual cues rather than from either modality alone. Most existing multimodal classifiers process each sample independently and rely on feature fusion for prediction, which limits their ability to exploit the statistical relationships among related samples in the dataset. To address this limitation, this work proposes a multi-graph contextual classification architecture that models dataset-level relational structure through three complementary graphs: a text semantic graph, a visual semantic graph, and a class-discriminative graph. The text and visual graphs capture linguistic and image-level neighborhood similarity using compact representations derived from BERT and CLIP embeddings, while the class-discriminative graph uses a Linear Discriminant Analysis projection to emphasize class-aware proximity in a shared multimodal space. Each graph is processed with a Graph Attention Network to enable adaptive neighborhood aggregation and context-aware feature propagation. The resulting graph-specific representations are then combined for final classification. The proposed framework shifts multimodal classification from isolated sample-level prediction to relational inference over multimodal statistical neighborhoods. This design offers a hybrid statistical and deep learning pipeline that integrates pretrained encoders, dimensionality reduction, discriminative projection, and graph-based reasoning to improve contextual understanding in multimodal meme analysis.
